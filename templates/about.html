About Production Analyser

Production Analyser is a comprehensive web application designed to provide detailed analytics and insights into production processes.

This tool helps monitor and analyze key performance indicators such as cycle times, fault delays, and tip dress efficiency across various production lines.

Features

  • Real-time dashboard with key metrics
  • CycleTime Backup Analytics
  • FaultDelay Backup Analytics
  • TipDress Backup Analytics
  • Interactive data visualization
  • Dark/Light theme support
  • Mobile-responsive design

Technology Stack

  • Backend: Flask (Python)
  • Frontend: HTML, CSS (Tailwind CSS), JavaScript
  • Data Processing: OpenPyXL for Excel file handling
  • Styling: Tailwind CSS with dark mode support

Built with modern web technologies to ensure fast, reliable, and user-friendly experience.
